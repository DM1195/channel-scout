# Channel Intel Schema

One file per channel: `channel-intel/linkedin.md`, `channel-intel/substack.md`, etc.

## Required Sections

### Platform
Name of the platform.

### Top Performers
List of 3-5 posts/videos/threads that performed best. For each:
- Creator handle
- Format/structure description (NOT full text — max 50 words)
- Engagement metric
- What made it work (1 sentence)

### Winning Template
The repeatable structure of top content on this channel:
- Hook pattern
- Body structure
- Length/format
- Cadence of top performers

### Gap Analysis
What is missing or underserved in this channel for this niche. 1-3 sentences.

### Saturation Score
Rating 1-5 (1 = wide open, 5 = extremely crowded) with one-line justification.

### Fit Signal
Is the founder's ICP active here? Evidence (subreddits found, subscriber counts on niche newsletters, etc.)

### Last Updated
Date.
